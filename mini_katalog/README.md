# Mini Katalog Uygulaması

Bu proje, Flutter kullanılarak geliştirilmiş temel bir katalog ve e-ticaret simülasyonudur. Eğitim kapsamında istenen widget yapısı, sayfa geçişleri, lokal asset yönetimi ve basit state (sepet) yönetimi içermektedir.

**Proje Detayları:**
* **Kullanılan Teknolojiler:** Flutter & Dart
* **Paketler:** Sadece temel `material.dart` paketi kullanılmıştır.

**Çalıştırma Adımları:**
Projeyi web tarayıcısında test etmek için terminal dizininde şu komutu kullanabilirsiniz:
`flutter run -d chrome`
